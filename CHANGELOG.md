# Changelog

All notable changes to FineFlow are documented in this file.

## [1.0.0] - 2026-01-06

### Phase 1: Foundation
- User authentication (login, register, logout)
- User profile management
- Project CRUD operations
- Audit logging system
- Responsive dashboard layout

### Phase 3: Document Management
- File upload with drag-and-drop
- Document storage via Supabase Storage
- File preview (images, text)
- Signed URL downloads
- Soft delete functionality

### Phase 4: Text Extraction
- PDF text extraction
- Office document parsing
- HTML/Markdown processing
- Plain text handling

### Phase 5: AI Embeddings
- Vector embeddings generation
- Document summarization
- Chunk-based processing
- OpenAI integration

### Phase 6: Semantic Search
- Natural language search
- Similarity-based results
- Project filtering
- File type filtering

### Phase 7: Admin System
- Admin dashboard
- User management
- System statistics
- Role-based access control

### Phase 8: Export System
- JSON export
- CSV export
- Plain text export
- Batch export support

### Phase 9: Quota Enforcement
- Tier-based limits (Free, Starter, Pro, Enterprise)
- Document count limits
- Storage limits
- Monthly processing limits
- Real-time quota tracking
- Upgrade prompts

### Phase 10: Notifications
- In-app notification system
- Processing status alerts
- Quota warnings
- Mark as read functionality
- Notification bell with badge

### Phase 11: Testing & Documentation
- Vitest testing framework setup
- Service unit tests
- API documentation
- Deployment guide
- User guide
- Architecture documentation
